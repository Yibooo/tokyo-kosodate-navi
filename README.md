# 東京23区・子育て支援ナビゲーター

東京23区在住の妊婦・子育て世帯向けに、国・都・区の子育て支援制度を一元検索・表示するWebアプリです。

## 概要

- 居住区・子の年齢・世帯年収などを入力するだけで、受給できる制度を自動判定
- 国・東京都・区の3層構造で制度を整理・表示
- 申請漏れを防ぎ、受給総額を可視化

## 対応区（MVP）

港区 / 品川区 / 目黒区 / 江東区 / 江戸川区
